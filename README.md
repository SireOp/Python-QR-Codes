# Python-QR-Codes
Quick QR Codes
Run main.py to crate a QR code
Or run "python bar.py" to make a png file named "barcode_sheet". When prompted each String you enter (Copy & Paste to prevent typos) is saved to the barcode_sheet.png, which is created once you type done.
pil library used for the QR codes, and for Simple strings the library pillow was used for the barcodes.
